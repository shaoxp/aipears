# OpenClaw 混合版（技术+非技术都能看）

> 用途：同一份内容同时面向“普通读者 + 技术读者”

## 使用方法
- 正文先讲生活化比喻（80%人能听懂）
- 每一节末尾加一个“技术补充框”（给20%技术读者）
- 视频里主讲通俗内容，技术补充放评论区/置顶文案

---

## A. 混合版文章结构（可直接按此重写）

1. 开场问题：AI会说但不会稳定做事
2. 生活比喻：餐厅店长（OpenClaw）+ 员工（Agents）
3. 四步工作法：听懂 -> 拆解 -> 执行 -> 检查
4. 真实场景：内容生产/学习整理/职场文档
5. HAT分工：人定方向，AI提速度
6. 技术补充（简版）：
   - Agent 像“独立执行单元”
   - OpenClaw 像“编排系统”
   - 关键是流程可控、失败可恢复、结果可追踪
7. 结尾金句 + 行动建议

---

## B. 混合版视频节奏（3-5分钟）

- 0:00-0:20 钩子（痛点）
- 0:20-1:10 比喻（餐厅）
- 1:10-2:10 四步机制（通俗）
- 2:10-3:00 三个场景（普通人）
- 3:00-4:00 技术补充30秒（Agent≈执行单元，OpenClaw≈编排层）
- 4:00-4:30 HAT总结与行动建议

---

## C. 混合版一句话模板（可复用）

- 通俗句：OpenClaw 是 AI 团队的“店长系统”。
- 技术句：OpenClaw 是多 Agent 的编排与治理层。

- 通俗句：AI 负责跑腿，人负责拍板。
- 技术句：Human-in-the-loop 决定系统安全边界与最终决策权。
